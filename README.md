# OnClass E2E Tests

* HTTP
* Typescript
* Jest
* Axios

## Installation

```bash
git clone git@github.com:pvolyntsev/o-tests.git
cd o-tests
npm install
```

## Run tests

```bash
npm run test:jest
```
